# 硬件接线



## 接线示意

接线前请确保断电操作。

### 机械臂

- 机械臂串口连接至 `/dev/ttyACM0`
- 波特率：115200

### 电机

| 电机 | GPIO Chip | Channel |
|------|-----------|---------|
| 左电机 | 4 | 0, 1 |
| 右电机 | 4 | 2, 3 |

### 摄像头

- USB 接口，设备文件：`/dev/video0`
